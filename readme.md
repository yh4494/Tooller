Tooller 专为个人打造的博客管理系统

> 详细请见 [Tooller](http://120.53.124.65/book/show/39)，闲余时间做的一个算是工具吧！有兴趣可以加我QQ。

#### 环境
PHP7.0+

Nginx/Apache

MySQL

#### 安装

~~~
composer install
php artisan key:generate
php artisan serve
~~~

##### .env配置

~~~bash
php artisan key:generate
INVITATION_CODE=SOMEAONEWASALLENS
~~~
