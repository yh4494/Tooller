ORACLE SQL 整理大全

oracle查询某用户下的所有表及视图

  1. -- 查询所有用户的表,视图等。
  2. select * from all_tab_comments
  3. -- 查询本用户的表,视图等。
  4. select * from user_tab_comments
  5.  --查询所有用户的表的列名和注释。
  6. select * from all_col_comments
  7. -- 查询本用户的表的列名和注释。
  8. select * from user_col_comments
  9. -- 查询所有用户的表的列名等信息。
  10. select * from all_tab_columns
  11. --查询本用户的表的列名等信息。
  12. select * from user_tab_columns

ORACLE下有三个视图
DBA_TABLES 拥有DBA角色的用户可以查看系统中的所有表
USER_TABLES 登录数据库的当前用户拥有的所有表
ALL_TABLES 登录数据库的当前用户有权限查看的所有表

ORACLE创建、删除索引等操作

  1. -- 创建索引
  2. create index 索引名 on 表名(列名);
  3. -- 删除索引
  4. drop index 索引名;
  5. -- 创建组合索引
  6. create index 索引名 on 表名(列名1,,列名2);
  7. -- 查询索引
  8. -- 根据索引名，查询表索引字段
  9. select * from user_ind_columns where index_name='索引名';
  10. -- 根据表名，查询一张表的索引
  11. select * from user_indexes where table_name='表名';
  12. -- 查询某一个索引
  13. select* from all_indexes where table_name='ACM_NETWORK_OPERATION';
  14. select * from user_ind_columns where table_name='ACM_NETWORK_OPERATION';

Sequence

创建一个Sequence

  1. create sequence MYUSER.S_MYSEQUENCE
  2. minvalue 1
  3. maxvalue 999999999999999999999999999
  4. start with 1--起始值
  5. increment by 1--增量
  6. cache 20;

删除Sequence

  1. DROP SEQUENCE MYUSER.S_MYSEQUENCE;

查询下一个值

  1. SELECT  SEQ_NAME.NEXTVAL FROM dual;

修改下一个值

  1. alter sequence SEQ_NAME increment BY X;    ----X为正数负数都可以

重新将序列的值的增长值设为1

  1. alter sequence SEQ_NAME increment BY 1;

在创建sequence时有个cache，比如cache 20 会先创建20个序列，以此来提高效率，要不然会每次插入数据时都去创建你需要的id，当经常插入数据时效率较低，所以oracle有这种机制，但是当你关闭数据库时，那20个序列号没用完，下次再插入数据时，他会从21开始创建，所以会不连续，

  1. create sequence SEQ_EMP_NO
  2.     minvalue 10001
  3.     maxvalue 99999999
  4.     start with 10001
  5.     increment by 1
  6.     cache 20;
